Base Classes
===============

The following section outlines the base interfaces. These provide
the base interface which should be inherited by all implementations.

.. warning::

    These are not designed to be used directly. They are to be used
    as base classes for implementations. To see the official implementation
    of defectio, see :mod:`defectio.api`.
